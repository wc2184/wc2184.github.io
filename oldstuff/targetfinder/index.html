Bitcoin Target Calculator

This tool takes the latest block and provides it's block hash (answer to the cryptographic puzzle) as well as the target hash and gives you a reference for just how small the target actually is and how difficult it is to find it.

What exactly was the target hash that bitcoin miners worked so hard to try to find a hash value that was equal or below to?

In other words, how many goddamn zeros/just how low is the number of the cryptographic puzzle that bitcoin miners work so hard to try to generate a hash below?
